SUI NFT Battle

sui-nft-battle.vercel.app

![Sui Logo](https://blog.sui.io/content/images/2023/04/Sui_Droplet_Logo_Blue-1.png)

**SUI NFT Battle** is a high-performance Hybrid Decentralized Application (dApp) that bridges the speed of the **Sui Blockchain** with the social interactivity of **Supabase**. Users can forge (mint) unique NFTs, showcase their armory, and engage in a community-driven battle for the "Weekly Champion" title based on likes.

---

Core Features

NFT Forge Station**: Seamlessly mint unique NFTs directly onto the Sui Network.
Dual Media Support**: Flexibility to use external image URLs or upload files directly from your local gallery to Supabase Storage.
Battle Feed & Leaderboard**: A real-time arena where the most liked NFTs rise to the top of the leaderboard.
Social Engagement**: Integrated comment system for each NFT and a secure anti-spam like mechanism.
Modern Responsive UI**: Optimized for all devices featuring a sleek "Micro-Armory" horizontal slider for mobile users.

---

Tech Stack

* **Frontend**: React.js, Vite, Tailwind CSS.
* **Blockchain**: Sui Network (Move Smart Contracts).
* **Backend/Database**: Supabase (PostgreSQL, Storage, & Real-time API).
* **Wallet Integration**: Mysten Labs dApp Kit.

Getting Started

1. Prerequisites
* Node.js (v18 or later)
* Sui Wallet browser extension
* Supabase Account

2. Local Installation
```bash
# Clone the repository
git clone [https://github.com/your-username/sui-nft-battle.git](https://github.com/your-username/sui-nft-battle.git)

# Navigate to the UI directory
cd sui-nft-battle/ui

# Install dependencies
npm install
